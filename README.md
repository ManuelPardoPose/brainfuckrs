# brainfuckrs - *Simple Interpreter for the brainfuck language*

## Does the following
- Interprets the brainfuck code in the specified file
- Input and output to the program is given via stdio

## `brainfuckrs -h`
```
A Simple Brainfuck Interpreter

Usage: brainfuckrs <FILE>

Arguments:
  <FILE>  Path to the Brainfuck file

Options:
  -h, --help  Print help
```

## Install
`cargo install --git https://github.com/ManuelPardoPose/brainfuckrs.git`
